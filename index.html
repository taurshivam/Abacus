How to Use an Abacus

Learn the basics of abacus counting!

🔴

Upper Bead = 5

Each upper bead is worth 5. Move it down toward the middle bar to count it.

🔵

Lower Bead = 1

Each lower bead is worth 1. Move it up toward the middle bar to count it.

🏛️

Place Values

Each column represents a place: ones (1s), tens (10s), hundreds (100s), and more!

🔢

Reading a Number

Add up all the active beads in each column. Right column = 1s, next = 10s, next = 100s.

💡

Example: 7

Move 1 upper bead (=5) + 2 lower beads (=2) = 7. Try it in Free Play!

🧮

History

Abacuses are over 2,000 years old! Ancient people used them to do math long before calculators.
